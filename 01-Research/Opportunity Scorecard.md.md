# Opportunity Scorecard

## Propósito

Este documento permite evaluar de manera objetiva las oportunidades identificadas para Nebo Flow.

El objetivo es priorizar aquellas que generan mayor valor para el cliente, tienen un mercado atractivo y pueden validarse rápidamente mediante un MVP.

---

# Metodología

Cada oportunidad será evaluada utilizando una escala de **1 a 5**, donde:

| Puntuación | Significado |
| ---------- | ----------- |
| 1          | Muy bajo    |
| 2          | Bajo        |
| 3          | Medio       |
| 4          | Alto        |
| 5          | Muy alto    |

La evaluación debe basarse en entrevistas con clientes, investigación de mercado y experiencia del equipo, evitando decisiones únicamente por intuición.

---

# Criterios de evaluación

| Criterio               | Peso | Pregunta                                       |
| ---------------------- | ---: | ---------------------------------------------- |
| Dolor del cliente      |  20% | ¿Qué tan grave es el problema?                 |
| Frecuencia             |  15% | ¿Con qué frecuencia ocurre?                    |
| Disposición a pagar    |  20% | ¿Existe presupuesto para resolverlo?           |
| Facilidad para validar |  10% | ¿Podemos entrevistar clientes fácilmente?      |
| Tamaño del mercado     |  15% | ¿Cuántas organizaciones tienen este problema?  |
| Complejidad del MVP    |  10% | ¿Podemos construir un MVP en pocas semanas?    |
| Escalabilidad          |  10% | ¿Puede crecer a múltiples industrias o países? |

# Criterios para avanzar

Una oportunidad será priorizada cuando:

* Resuelva un problema frecuente.
* Tenga un comprador claramente identificado.
* El costo del problema sea significativo.
* Exista disposición a pagar.
* Sea posible validar el MVP rápidamente.
* Tenga potencial para escalar.
